# TODO List App

A **simple and functional TODO List App** built with **HTML, CSS, and JavaScript**.  
Perfect for beginners learning vanilla JavaScript and creating small, real-world web projects.

---

## Features
- Add new tasks  
- Mark tasks as complete  
- Save tasks using **LocalStorage**  
- Instant updates on the interface  

---

## Technologies Used
- **HTML** – structure of the app  
- **CSS** – styling and layout  
- **JavaScript** – app logic and LocalStorage handling  

---

## How to Use
1. Open `index.html` in your browser  
2. Type a task in the input box  
3. Click **Add** to save it  
4. Check the box to mark a task as completed
